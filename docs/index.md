# ScuffedHolograms

ScuffedHolograms is a hologram plugin for Hytale servers. It renders floating text, images, GIFs, 3D models, and particle effects in the world. Everything is created and edited from an in-game GUI.

Holograms can show live data (online count, time, leaderboards, player stats), respond to clicks or to players walking into them, and run commands on interaction. A single hologram can act as a sign, a leaderboard, a shop button, or a quest-giving NPC.

---

## What you can build

- **Text holograms.** Multi-line text with colours, gradients, rainbows, and per-line animations.
- **Image and GIF holograms.** Upload a PNG, JPG, or BMP, or an animated GIF, and display it in the world.
- **Model holograms.** Display any in-game 3D model asset.
- **Particle-effect holograms.** Anchor a particle effect at a location.
- **Leaderboards.** Top-N rankings for play time, joins, and supported level plugins.
- **Live signs.** Text that updates itself: clock, date, online count, per-player stats.
- **Interactive holograms.** Run commands when a player clicks the hologram or walks into it, with chance rolls, cooldowns, per-player limits, and permission gates.
- **Quest-giver NPCs.** Ordered actions gated behind quest completion, with a supported quest plugin.
